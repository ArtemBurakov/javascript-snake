# javascript-snake

Simple snake game, developed as my first attempt at using canvas and javascript classes.

# Dependencies

This game does not require any dependencies.

# Installation

Extract all files from downloaded folder. Run `index.html` file.

# Usage:
## Controls:

- start: `space`
- top: `↑` - up arrow
- down: `↓` - down arrow
- left: `←` - left arrow
- right: `→` - right arrow
